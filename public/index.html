INÍCIO DE 2024
Primeiros passos com Front-End

Comecei pela parte mais visual da web — HTML, CSS e JavaScript. Aprendi a construir layouts, trabalhar com responsividade e entender como um navegador processa páginas. Foi o ponto de partida que me fez querer ir mais fundo.

HTML CSS JavaScript
MEADOS DE 2024
A virada: Back-End

O front me interessou por um tempo, mas quando comecei a entender o que acontecia do outro lado — no servidor — que realmente me apaixonei. APIs, banco de dados, fluxo de requisições. Mudei o foco e não parei mais.

APIs REST PostgreSQL Java Python
FINAL DE 2024
Segurança da Informação

Comecei a estudar segurança em paralelo ao back-end. Redes, criptografia, vulnerabilidades e proteção de dados. Quanto mais entendia como sistemas funcionam, mais queria saber como eles falham — e como protegê-los.

Redes Criptografia LGPD Cyber
